# PokeIndex

Simple Vue 3 app using [PoKeApi](https://pokeapi.co/) imitating Pokedex.

## Developing

Run backend server

```shell
cd backend
npm run install
npm run serve
```

then run frontend server

```shell
cd frontend
npm run install
npm run dev
```

## Runing the app

Run `docker compose up --build` to spin up the docker container. App will be exposed on [localhost:8080](http://localhost:8080)
